AvidXchange Unified UI

Project Overview

In the 18 years since Avidxchange's founding, it has created many applications including tools for managing invoices, payments, utility usage, etc. These applications were developed over time and have different user interface styles, layouts, and affordances. The inconsistency between applications has been a source of user issues and complaint. I was tasked with unifying the AvidXchange suite of products into a cohesive experience.


After conducting focus groups, user feedback sessions and digging through support tickets, the biggest issues with users were apparent: Inconsistent navigation, vastly different look and feels between the applications, having different login experiences, and a general disjointed experience.

We chose a solution that would address user concerns with minimally affecting the refactoring of the applications (and thus affecting training, documentation and support). This included a global navigation, a robust theming and reskinning ability, and adding some highly requested features.
